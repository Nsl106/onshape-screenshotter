# Onshape Progressor

Render a thumbnail of your Onshape part studio or assembly on a schedule and commit
it as a timelapse frame, so your FRC team can watch its CAD progress over a season.

Each team runs its own private copy — your API keys and rendered images never leave
your repository, and no central server ever sees them.
